# Cash Burn Analysis

This project analyzes company cash burn patterns to identify trends, cost drivers, and potential financial risks.

## What it does
- Tracks and analyzes monthly cash burn over time
- Identifies trends and changes in spending patterns
- Highlights key cost drivers impacting burn
- Flags potential financial risks based on burn behavior
- Generates executive-level insights using AI

## Tools used
- Zapier
- Google Sheets
- AI prompting
